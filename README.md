## Setup

-   Run `python3 -m venv venv`
-   Run `source ./venv/bin/activate`
-   Run `pip install pyspark`
-   Run `pip install delta-spark`
-   Run `pip install numpy`
-   Run `python3 ./example.py`
-   Run `python3 ./main.py`
